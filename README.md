# uni-app_shop

## 项目介绍

`uni-app_shop`是一个购物类的前端项目，基于uni-app+vue实现。主要首页、分类、购物车、搜索、商品详情、结算等功能。

### 技术选型

| 技术                        | 说明                           |
| --------------------------- | ------------------------------ |
| uni-app                     | 使用 vue开发所有前端应用的框架 |
| vue                         | 前端框架                       |
| vuex                        | 组件间通信                     |
| @escook/request-miniprogram | 第三方发起网络数据请求包       |
